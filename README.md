# 🎉 Convite - Sonho de Menina 🎉

Este é um projeto de site para criar um convite digital para um evento de 15 anos, com animações e uma estrutura responsiva para se ajustar a diferentes dispositivos. 🌸

## Descrição 📝

O site apresenta um layout de convite para o evento "Sonho de Menina", incluindo imagens decorativas, informações sobre o evento, um mapa do local e interações para navegação entre as seções. A estrutura é responsiva e adapta-se bem a dispositivos móveis, tablets e desktops. Além disso, o site inclui animações de pétalas caindo no fundo para um efeito visual mais dinâmico. 💐

## Funcionalidades ⚙️

- **Navegação entre seções**: Links para navegar pelas seções de Participantes e Informações. 🔗
- **Imagens dinâmicas**: Exibição de imagens que se ajustam ao tamanho da tela. 🖼️
- **Mapas incorporados**: Exibição do local do evento usando o Google Maps. 📍
- **Animações de fundo**: Pétalas caindo com animações. 🌸
- **Responsividade**: O layout é adaptável a diferentes tamanhos de tela, incluindo dispositivos móveis. 📱💻

## Tecnologias 🛠️

- **HTML5**: Estrutura semântica para a página. 📄
- **CSS3**: Estilos e animações para o design da página. 🎨
- **JavaScript**: Animações e interatividade (como o efeito de pétalas caindo e transições de scroll). ✨
- **Google Fonts**: Uso de fontes externas para melhorar o design tipográfico. 🅰️🅱️
